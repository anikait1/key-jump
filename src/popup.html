Keyboard Hints

Vimium-style navigation for any website

Hint Mode (Common)

;Show click hints
Shift + ;Right-click hints

Hint Mode (All Elements)

Ctrl + ;Show all click hints
Ctrl + Shift + ;All right-click hints

While in Hint Mode

a-zType hint characters
BackspaceDelete last character
EscCancel

When a context menu is open, hints automatically scope to menu items only.
